Constraint programming

Relations, not procedures

Addition procedure

		123
		+ 4
		---

Addition relation

Forwards: 2 + 3 = x
Backwards: x + 3 = 5
Relation: x + y = 5
Test: 2 + 3 = 6 => false

Cassowary

  • Only works on linear equations
  • API designed for UI applications
  • Very fast
  • Alex Russell's fork works in node or the browser.

Cassowary API

		var solver = new c.SimplexSolver();
		var spending = new c.Variable({ name: "spending", value: 3598 });
		var revenues = new c.Variable({ name: "revenues", value: 2328 });
		var deficit = new c.Variable({ name: "deficit"});
		var balance = new c.Equation(c.minus(spending, revenues), deficit, c.Strength.required, 0),
		solver.addConstraint(balance);
